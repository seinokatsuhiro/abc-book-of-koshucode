# 甲州記法の ABC


この本は、甲州記法というデータの表記法に関する入門書です。
データの読み・書き・計算の体系を、
統合したひとつの体系として学べるようになっています。
本全体は [zip ファイル][zip] として入手できます。
例題をコンピュータで実行しながら学習することをおすすめします。
実行結果は [入出力リストの要約][sum] に一覧されています。
